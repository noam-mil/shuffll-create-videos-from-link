Technical notes

  • Route: /video-generator (standalone, no auth)
  • Trigger: POST to a Google Apps Script web app URL (stored as a constant)
  • Response: JSON with videoUrl field
  • i18n: uses existing useTranslation() hook + EN/HE toggle
  • Video player: reuses existing CloudflareVideoEmbed or plain <video> tag depending on URL type
