Design decisions

  • Brand orange gradient (#f97316 → #ff8c00) on buttons, step badges, and glows — matches the app's --primary + --accent tokens
  • Warm dark background (hsl(30 15% 8%)) — same dark mode as the rest of the app
  • Step 1 collapses to a slim "done" row in state 3 so the video has full breathing room
  • Language toggle (EN / עב) top-right using useTranslation() — same i18n setup already in the app
  • Generate button disabled until URL has content, orange glow on active step card
  • Reset (↺) button returns to idle state so the user can generate another video
